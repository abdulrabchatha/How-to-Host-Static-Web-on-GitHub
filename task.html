How to Host Static Web Page on Git-Hub

  1. Make a new folder.
  2. Now, open Command Prompt in that Folder.
  3. Run command "Code ." to open visual code in that folder.
  4. Make a new File of Html extension
  5. It looks like this now..
  6. Open Terminal and Run command "git init". It will a create a hidden folder that contains Git Files.
  7. Now run command "git status", it will tell us which files are to be added in git hidden folder.
