How to Host Static Web Page on Git-Hub

  1. Make a new folder.
  2. Now, open Command Prompt in that Folder.
  3. Run command "Code ." to open visual code in that folder.
  4. Make a new File of Html extension
  5. It looks like this now..
  6. Open Terminal and Run command "git init". It will a create a hidden folder that contains Git Files.
  7. Now run command "git status", it will tell us which files are to be added in git hidden folder. Files to be added marked in red color.
  8. Run "git add ." or "git add 'Filename'" to add files or specific file. Again check status of files using "git status". Now files are added and marked in green color.
  9. Commit your work using "git commit -m "'any message'"
  10. Make a repository on Git-Hub and copy the Address.Run the command "git remote add origin " 'Link Address' " "
  11. In last, push the data from hidden Git folder to remote repository using "git push". at first you get error of this type.
  12. Solution to above error is given. Simply copy and paste it.

Branches Concept

  1. "git branch" is used to see the available branches
  2. "git branch chatha" used to create a new branch named "chatha"
  3. "git checkout chatha" used to change your branch from origin/master to chatha
  4. "git merge chatha" used to merge brnch named chatha with you are currently active
  5. "git branch -d chatha" used to delete branch "chatha" after merging it.
  6. "git branch -D chatha1" used to delete branch "chatha1" without merging it.

Cloning Concept

  1. "git clone 'repo link' " is used to clonr the repository folder on other pc
  2. Now, I added a new style.css file in clone folder and checking its status
  3. Adding the untracked files to the git local hidden folder using "git add ."
  4. Pushing the local git data to online repository using push command after commiing it.
  5. When I try to push local git files to online repository got error of pulling data from online repository because there exists a file that is not availabe in first pc.
  6. To solve this run command "git pull"

Now your files are added to your Git-Hub Repository. To make web live follow NExt Steps...

  1. Go to Settings. Make Sure that project repository in Public View.
  2. Now go to Pages
  3. Select Source "Deploy from Branch". And Branch is "master" and "/(root)" like in the picture. Click Save...
  4. In couple of minutes you got the Link your Live Web Page.
